# PoE2 Trade Import

Full port of the PoE2 Trade Extension. Adds an "Import Item" button to the trade site. Fixes duplicates and missing stats.

## Features
- **Import Item**: Adds an "Import Item" button to the trade site to easily import item stats.
- **Fixes**: Handles duplicates and missing stats during import.

## Installation (Tampermonkey)
1. Install the [Tampermonkey extension](https://www.tampermonkey.net/) for your browser.
2. Click "Create a new script" in Tampermonkey.
3. Copy the contents of `PoE2-Trade-Import.js` into the editor.
4. Save the script.
5. Visit [Path of Exile 2 Trade](https://www.pathofexile.com/trade2/) and enjoy enhanced trading!
